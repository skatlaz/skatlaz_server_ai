Skatlaz Server AI 2.0 Beta

Django-based generative AI server with administration, RAG, agents, Ollama, Google Gemini, WebDiver and LoRA/QLoRA package generation for Ollama export.

Architecture

User question -> Agent -> RAG/WebDiver -> LLM provider (Ollama or Gemini) -> Database logs -> Answer.

Administration

The Django Admin manages LLM providers, models, prompt templates, agent categories, agents, RAG collections, documents, chunks, WebDiver tasks, datasets, LoRA/QLoRA jobs and chat sessions.

RAG

Upload TXT, CSV, JSON, HTML, PDF and DOCX as DocumentSource. Then index the source through /ai/rag/index-source/ or python manage.py skatlaz_index_rag.

WebDiver

Create tasks with URL and type _text, _html, _filter or _download. The result is stored and can feed a RAG collection.

Ollama + Gemini

Ollama is used for local inference. Gemini is used as a fallback or stronger model through the GOOGLE_API_KEY environment variable.

LoRA/QLoRA

Django generates a QLoRA training script and an Ollama Modelfile. Real training should run in a worker or separate GPU machine.

python manage.py skatlaz_prepare_finetune 1
ollama create model_name -f models/ollama_exports/job/Modelfile

Endpoints

  • POST /ai/chat/
  • GET /ai/models/list/
  • GET /ai/rag/search/
  • POST /ai/crawler/start/
  • POST /ai/finetune/prepare/
  • POST /ai/ollama/export/

Install

pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py skatlaz_seed_ai
python manage.py runserver

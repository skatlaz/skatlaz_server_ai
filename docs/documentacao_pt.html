Skatlaz Server AI 2.0 Beta

Servidor Django para IA generativa com administração, RAG, agentes, Ollama, Google Gemini, WebDiver e geração de pacotes LoRA/QLoRA para exportação ao Ollama.

Arquitetura

Pergunta do usuário -> Agente -> RAG/WebDiver -> Provedor LLM (Ollama ou Gemini) -> Logs no banco -> Resposta.

Administração

O painel Django Admin gerencia provedores LLM, modelos, templates de prompt, categorias de agentes, agentes, coleções RAG, documentos, chunks, tarefas WebDiver, datasets, jobs LoRA/QLoRA e sessões de chat.

RAG

Faça upload de TXT, CSV, JSON, HTML, PDF e DOCX como DocumentSource. Depois indexe a fonte pelo endpoint /ai/rag/index-source/ ou pelo comando python manage.py skatlaz_index_rag.

WebDiver

Crie tarefas com URL, tipo _text, _html, _filter ou _download. O resultado é registrado e pode alimentar uma coleção RAG.

Ollama + Gemini

Ollama é usado para inferência local. Gemini é usado como fallback ou modelo forte via variável de ambiente GOOGLE_API_KEY.

LoRA/QLoRA

O Django gera um script de treinamento QLoRA e um Modelfile do Ollama. O treinamento real deve rodar em worker ou máquina GPU separada.

python manage.py skatlaz_prepare_finetune 1
ollama create nome_do_modelo -f models/ollama_exports/job/Modelfile

Endpoints

  • POST /ai/chat/
  • GET /ai/models/list/
  • GET /ai/rag/search/
  • POST /ai/crawler/start/
  • POST /ai/finetune/prepare/
  • POST /ai/ollama/export/

Instalação

pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py skatlaz_seed_ai
python manage.py runserver

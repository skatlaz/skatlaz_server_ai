ELLM
POST /ai/mcp-task/
{
  "task_type": "ask",
  "prompt": "Explique meu documento",
  "language": "python",
  "target_os": "linux",
  "agent_slug": "rag-agent"
}

O endpoint recebe JSON, identifica o tipo de tarefa e envia para o runtime central. O runtime escolhe o agente, consulta RAG quando necessário e chama o provedor LLM.

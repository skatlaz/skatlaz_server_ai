MCP Registry

Documentation des MCP existants

Cette page catalogue les packages MCP attachés à Skatlaz Server 2.0. Chaque MCP peut fonctionner comme serveur FastAPI séparé, exposer sa documentation sur /docs et être appelé par Skatlaz Server via Agent Router ou MCP Runtime.

Skatlaz Client Ask
 → Skatlaz Server 2.0 /ai/mcp-task/
 → MCP Runtime / Agent Router
 → MCP spécifique : Office, Money, Music, SurfTask, GexProg...
 → réponse JSON structurée
 → Chat / Dashboard / Rapport

Comment orchestrer les MCPs dans Skatlaz Server

Utilisez agent_slug pour sélectionner le domaine de l’agent et task_type pour indiquer l’action. Le serveur peut répondre directement avec LLM/RAG ou transférer à un MCP spécialisé.

{
  "task_type": "code",
  "prompt": "crée une landing page en HTML",
  "language": "html",
  "target_os": "windows",
  "agent_slug": "skatlaz-gexprog"
}

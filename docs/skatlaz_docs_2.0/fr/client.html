Skatlaz Client 2.0 / Ask

Le Client doit rester simple : textarea de prompt, sélecteurs de task type, language, target OS et agent slug. La réponse revient en JSON ou texte dans le chat.

fetch('/ai/mcp-task/', {
  method: 'POST',
  headers: {'Content-Type':'application/json'},
  body: JSON.stringify({
    task_type: 'ask',
    prompt: prompt,
    language: 'python',
    target_os: 'all',
    agent_slug: 'skatlaz-client-agent'
  })
})

Flux du chat

1Utilisateur écrit
2Client envoie JSON
3Server orchestre
4LLM répond
5Chat affiche

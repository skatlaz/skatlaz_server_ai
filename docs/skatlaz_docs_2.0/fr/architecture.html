ELLM
POST /ai/mcp-task/
{
  "task_type": "ask",
  "prompt": "Explique meu documento",
  "language": "python",
  "target_os": "linux",
  "agent_slug": "rag-agent"
}

L’endpoint reçoit du JSON, identifie le type de tâche et l’envoie au runtime central. Le runtime choisit l’agent, consulte RAG si nécessaire et appelle le fournisseur LLM.

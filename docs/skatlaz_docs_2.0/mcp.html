MCP Registry

Documentação dos MCPs existentes

Esta página cataloga os pacotes MCP anexados ao Skatlaz Server 2.0. Cada MCP pode rodar como servidor FastAPI separado, expor sua documentação em /docs e ser chamado pelo Skatlaz Server através do Agent Router ou do MCP Runtime.

Skatlaz Client Ask
 → Skatlaz Server 2.0 /ai/mcp-task/
 → MCP Runtime / Agent Router
 → MCP específico: Office, Money, Music, SurfTask, GexProg...
 → resposta JSON estruturada
 → Chat / Dashboard / Relatório

Como orquestrar MCPs no Skatlaz Server

Use o campo agent_slug para selecionar o domínio do agente e o campo task_type para indicar a ação. O servidor pode decidir se responde direto pelo LLM/RAG ou encaminha para um MCP especializado.

{
  "task_type": "code",
  "prompt": "crie uma landing page em HTML",
  "language": "html",
  "target_os": "windows",
  "agent_slug": "skatlaz-gexprog"
}

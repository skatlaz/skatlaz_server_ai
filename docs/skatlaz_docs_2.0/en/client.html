Skatlaz Client 2.0 / Ask

The Client should stay simple: a prompt textarea, task type, language, target OS and agent slug selectors. The answer returns as JSON or text in the chat.

fetch('/ai/mcp-task/', {
  method: 'POST',
  headers: {'Content-Type':'application/json'},
  body: JSON.stringify({
    task_type: 'ask',
    prompt: prompt,
    language: 'python',
    target_os: 'all',
    agent_slug: 'skatlaz-client-agent'
  })
})

Chat flow

1User writes
2Client sends JSON
3Server orchestrates
4LLM answers
5Chat renders

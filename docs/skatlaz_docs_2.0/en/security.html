Best practices

The server should be secure, fast and observable.

Security

Use CSRF when possible, token authentication for API, rate limits and per-user logs.

Performance

Use workers for WebDiver, indexing, embeddings and fine-tuning.

Observability

Log payload, agent, collection, execution time, error and approximate cost per call.

Golden rule

Skatlaz Server deve responder rápido no chat e mover processos longos para filas. O Client Ask deve ser leve, bonito e direto.

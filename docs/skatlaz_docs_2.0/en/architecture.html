POST /ai/mcp-task/
{
  "task_type": "ask",
  "prompt": "Explique meu documento",
  "language": "python",
  "target_os": "linux",
  "agent_slug": "rag-agent"
}

The endpoint receives JSON, identifies the task type and sends it to the central runtime. The runtime selects the agent, queries RAG when needed and calls the LLM provider.

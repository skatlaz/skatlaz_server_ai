Overview

Skatlaz Server 2.0 is a Django server for generative AI. It centralizes LLM providers, agents, RAG collections, document sources, chunks, WebDiver tasks, datasets and fine-tuning preparation.

Ask Client → Django API → MCP Runtime → Agent Router → RAG/WebDiver/Code → Ollama or Gemini → JSON response

MCP Registry

Existing MCP documentation

This page catalogs the MCP packages attached to Skatlaz Server 2.0. Each MCP can run as a separate FastAPI server, expose its documentation at /docs and be called by Skatlaz Server through the Agent Router or MCP Runtime.

Skatlaz Client Ask
 → Skatlaz Server 2.0 /ai/mcp-task/
 → MCP Runtime / Agent Router
 → Specific MCP: Office, Money, Music, SurfTask, GexProg...
 → structured JSON response
 → Chat / Dashboard / Report

How to orchestrate MCPs in Skatlaz Server

Use agent_slug to select the agent domain and task_type to indicate the action. The server can decide whether to answer directly with LLM/RAG or forward to a specialized MCP.

{
  "task_type": "code",
  "prompt": "create an HTML landing page",
  "language": "html",
  "target_os": "windows",
  "agent_slug": "skatlaz-gexprog"
}

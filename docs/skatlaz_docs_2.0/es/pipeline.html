Pipeline operativo

Upload/Crawler → Extractor → Cleaner → Chunker → Embeddings → Vector Store → Retriever → Prompt Builder → LLM → Logs

Para tarefas pesadas, use fila com Celery ou RQ. Crawler, indexação e fine-tuning não devem bloquear o processo web.

Buenas prácticas de RAG

Chunks

Use chunks entre 500 y 1200 tokens.

Metadatos

Guarde título, URL, idioma, fecha y colección.

Prompt

Nunca envíe el documento completo al LLM; recupere solo pasajes relevantes.

Skatlaz Client 2.0 / Ask

El Client debe ser simple: textarea para prompt, selector de task type, language, target OS y agent slug. La respuesta vuelve como JSON o texto en el chat.

fetch('/ai/mcp-task/', {
  method: 'POST',
  headers: {'Content-Type':'application/json'},
  body: JSON.stringify({
    task_type: 'ask',
    prompt: prompt,
    language: 'python',
    target_os: 'all',
    agent_slug: 'skatlaz-client-agent'
  })
})

Flujo del chat

1Usuario escribe
2Client envía JSON
3Server orquesta
4LLM responde
5Chat renderiza

MCP Registry

Documentación de MCPs existentes

Esta página cataloga los paquetes MCP anexados a Skatlaz Server 2.0. Cada MCP puede ejecutarse como servidor FastAPI separado, exponer su documentación en /docs y ser llamado por Skatlaz Server mediante Agent Router o MCP Runtime.

Skatlaz Client Ask
 → Skatlaz Server 2.0 /ai/mcp-task/
 → MCP Runtime / Agent Router
 → MCP específico: Office, Money, Music, SurfTask, GexProg...
 → respuesta JSON estructurada
 → Chat / Dashboard / Informe

Cómo orquestar MCPs en Skatlaz Server

Use agent_slug para seleccionar el dominio del agente y task_type para indicar la acción. El servidor puede decidir si responde directo con LLM/RAG o envía a un MCP especializado.

{
  "task_type": "code",
  "prompt": "crea una landing page en HTML",
  "language": "html",
  "target_os": "windows",
  "agent_slug": "skatlaz-gexprog"
}

POST /ai/mcp-task/
{
  "task_type": "ask",
  "prompt": "Explique meu documento",
  "language": "python",
  "target_os": "linux",
  "agent_slug": "rag-agent"
}

El endpoint recibe JSON, identifica el tipo de tarea y lo envía al runtime central. El runtime elige el agente, consulta RAG cuando sea necesario y llama al proveedor LLM.

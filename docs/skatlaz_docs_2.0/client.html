Skatlaz Client 2.0 / Ask

O Client deve ser simples: textarea para prompt, seletor de task type, language, target OS e agent slug. A resposta retorna como JSON ou texto no chat.

fetch('/ai/mcp-task/', {
  method: 'POST',
  headers: {'Content-Type':'application/json'},
  body: JSON.stringify({
    task_type: 'ask',
    prompt: prompt,
    language: 'python',
    target_os: 'all',
    agent_slug: 'skatlaz-client-agent'
  })
})

Fluxo do chat

1Usuário escreve
2Client envia JSON
3Server orquestra
4LLM responde
5Chat renderiza
